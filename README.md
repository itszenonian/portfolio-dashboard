# Portfolio Dashboard

A cloud-hosted personal portfolio dashboard that tracks DeFi positions on the SUI blockchain, Binance futures & crypto loans, and stock/ETF holdings — all in one live-updating web UI.

Hosted on a Google Cloud VM, served over HTTPS via Nginx + Let's Encrypt, and auto-refreshed every 30 minutes via cron.

---

## What It Tracks

### SUI Chain (Wallet 1 — `WALLET_954C`)
| Protocol | Position Type | What It Shows |
|---|---|---|
| **Aftermath Finance** | SUI/USDC 80/20 Weighted AMM | USDC + SUI amounts, LP share, pending afSUI rewards |
| **NAVI Protocol** | vSUI collateral / USDsui borrow | Collateral value, debt, LTV, health factor, liquidation price |
| **Suilend** | Multi-collateral lending | All deposits (sSUI, WAL, DEEP, etc.), borrows, health factor |
| **SteammFi** | WAL/USDC 50/50 LP (via Suilend isolated market) | WAL + USDC amounts from pool math |
| **Walrus** | WAL staking | Total WAL staked, number of staking positions |
| **CETUS** | xCETUS ve-lock | xCETUS balance → USD |
| **Scallop** | Supply + obligation | Receipt token balances, collateral deposits, borrows |
| **Ember Finance** | eBLUE yield vault | eBLUE amount, BLUE equivalent via on-chain exchange rate, yield % |

### SUI Chain (Wallet 2 — `WALLET_E64C`)
| Protocol | Position Type | What It Shows |
|---|---|---|
| **Aftermath Finance** | LBTC/lzWBTC 60/40 Weighted AMM | WBTC + LBTC amounts, LP share, pending DEEP rewards |
| **CETUS** | Concentrated Liquidity (CLMM) | SUI/USDsui position — token amounts, price range, in/out of range status |

### Ethereum (`WALLET_ETH`)
| Protocol | Position | What It Shows |
|---|---|---|
| **Zentry** | stZENT ERC-20 staking | stZENT balance → USD via CoinGecko price |

### Binance (CEX)
| Account | What It Shows |
|---|---|
| **Futures account** | Margin balance, wallet balance, unrealized PnL, all open positions (symbol, side, size, entry, mark, leverage) |
| **Crypto Loan** | Collateral coin + amount, loan coin + debt amount, current LTV, net value |

### Stocks & ETFs
| Market | What It Shows |
|---|---|
| **US stocks/ETFs** | Ticker, shares, avg cost, live price (Yahoo Finance), market value, gain %, daily change % |
| **Thai SET stocks** | Same as above with THB → USD FX conversion (auto-detects `.BK` suffix) |

## Dashboard Pages

### Overview
- **Stats bar** — Gross assets, Total debt, Net value, Position count
- **Category donut chart** — AMM / Lending / Staking / Futures / Stocks breakdown
- **Protocol bar chart** — Value ranked by protocol (Aftermath, Suilend, NAVI, Walrus, etc.)
- **Portfolio history line chart** — Weekly total, crypto, and stock values over time
- **Token summary tables** — Aggregated holdings (assets + debts) across all protocols, stock holdings, and by-category summary
- **Allocation bars** — % of portfolio per category with colored progress bars
- **Lending risk cards** — NAVI and Suilend health factor + LTV bars with zone coloring (safe / warning / danger)

### Cashflow (Interest Tab)
- **Stats** — Cumulative earned, avg per week, avg annualized yield, best week ever
- **Stacked bar chart** — Weekly Main + MM earnings over time
- **Annualized yield % line chart** — Weekly yield trend
- **Full breakdown table** — Every week: Main, MM, Total, Cumulative, Ann. Yield %, Crypto Portfolio value

### Positions
- **Position cards** grouped by category (AMM / Lending / Staking / Futures / Stocks)
- Each card shows: protocol badge, category badge, token amounts, LTV health bar (for lending), total USD value
- Click any card → modal with full token breakdown, debt breakdown, stats
- **Stock cards** — editable (click to update shares / avg cost in-place)
- **Add Stock button** — add new tickers on the fly

## Architecture

```
Browser (HTTPS)
     │
  Nginx (port 443)          ← SSL via Let's Encrypt (zenonian.duckdns.org)
     │
  server.py (port 8765)     ← Python BaseHTTPServer, HTTP Basic Auth
     │
  ┌──┴──────────────────┐
  │  portfolio_dashboard │  ← Static HTML generated by generate_dashboard.py
  │  .html               │
  └─────────────────────┘
     │
  generate_dashboard.py     ← Run by cron every 30 min (or /refresh button)
     │
  ┌──┬──────────────────────┬─────────────────────┬──────────────┐
  │  │                      │                     │              │
SUI RPC            Binance REST API       Yahoo Finance     CoinGecko
(BlockVision +     (Futures + Loan)       (Stock prices)    (Crypto prices)
 public fullnode)
```

### VM Setup
- **Platform:** Google Cloud Compute Engine (`e2-micro`, `asia-southeast1-a`)
- **OS:** Debian (Bookworm)
- **Service:** `systemd` keeps `server.py` running 24/7, auto-restarts on crash
- **Cron:** Regenerates dashboard every 30 minutes
- **HTTPS:** Nginx reverse proxy + Certbot (Let's Encrypt, auto-renews)
- **Auth:** HTTP Basic Auth on all routes (set via `.env`)

### Server API endpoints (`server.py`)
| Endpoint | Method | What It Does |
|---|---|---|
| `/` | GET | Serves `portfolio_dashboard.html` |
| `/refresh` | POST | Runs `generate_dashboard.py`, reloads page |

## Setup (for your own deployment)

### 1. Clone and configure
```bash
git clone https://github.com/itszenonian/portfolio-dashboard
cd portfolio-dashboard
cp .env.example .env
# Fill in your keys and wallet addresses in .env
```

### 2. Deploy to a GCE VM
```bash
# Copy files to your VM
gcloud compute scp ./* your-vm:/opt/portfolio/ --zone=your-zone

# Run setup
bash setup_vm.sh
```

### 3. Enable HTTPS
```bash
# Step 1: self-signed cert (works immediately)
bash setup_https.sh

# Step 2: real cert (after pointing a domain to your VM IP)
bash upgrade_to_letsencrypt.sh your-domain.duckdns.org
```

### Required `.env` keys
```
BLOCKVISION_API_KEY   — BlockVision SUI RPC API key
BINANCE_API_KEY       — Binance API key (Futures + Loan read access)
BINANCE_SECRET_KEY    — Binance secret key
DASHBOARD_USER        — Login username for the dashboard
DASHBOARD_PASS        — Login password for the dashboard
WALLET_954C           — Primary SUI wallet address
WALLET_E64C           — Secondary SUI wallet address
WALLET_ETH            — Ethereum wallet address
```

## Tech Stack

- **Backend:** Python 3 (stdlib only + `requests`, `python-dotenv`)
- **Frontend:** Vanilla HTML/CSS/JS — no frameworks
- **Charts:** Chart.js 4.4
- **Data sources:** SUI RPC (BlockVision + public fullnode), Binance REST API, CoinGecko, Yahoo Finance, DexScreener
- **Infrastructure:** GCE e2-micro VM, Nginx, Let's Encrypt (Certbot), systemd, cron
- **Weekly logging:** Google Sheets via `gspread`
